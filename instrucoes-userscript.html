Instruções para adicionar o userscript

A vantagem do userscript é ser universal: funciona no Chrome, Microsoft Edge, Opera Next, e Firefox.

No momento a experiência do usuário ainda é um pouco inferior à extensão nativa, mas todos os sites suportados pela extensão também funcionam com o userscript.

Seguem as instruções:

  1. Instale a extensão Tampermonkey
  2. Adicione o burlesco.user.js
  3. Limpe o cache do seu navegador (instruções)
  4. Pronto! Agora é só aproveitar o Burlesco!

Alternativamente, é possível instalar a extensão manualmente no Chrome.
